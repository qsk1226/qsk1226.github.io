java.utils.Collection 集合大礼包

集合表示了一组对象，而Collection是集合层次结构的根接口。一些集合可以存在重复的元素，而有些则不允许，一些是有序的，而又有一些是无序的。
jdk并没有提供此接口的直接实现，而是通过具体的子接口，例如List、Set接口来定义集合的实现。此接口通常使用在比较通用的场合来传递集合并操作它们。
