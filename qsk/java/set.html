Set详解

java 集合 2019/09/14

  • Collection集合

  • List集合 继承Collection接口，集合有序、允许重复值、可插入null。

  • Set集合 集合无序，不允许重复值，

  • Queue队列

Set 详解

Set继承自Collection接口,其实现并没有对Collection接口进行任何扩展。

  • Set集合中每个元素都是独一无二的，它不包含重复元素(e1.equals(e2)),可包含至多一个null元素。
  • Set集合不保证元素的有序性。
